## 🎨 Color Specification Guide

### **Header Background (Red)**
```
Hex:     #E8312B
RGB:     232, 49, 43
Usage:   Table headers, titles, company branding
Effect:  Professional red - conveys importance and financial matters
Contrast: White text on red background (AAA standard)
```

### **Summary Card Background (Light Gray)**
```
Hex:     #ECF0F1
RGB:     236, 240, 241
Usage:   Cell backgrounds, summary boxes
Effect:  Clean, neutral, professional appearance
Contrast: Dark text on light gray (AAA standard)
```

### **Debit Amount (Red Text)**
```
Hex:     #E8312B
RGB:     232, 49, 43
Usage:   Debit amounts, receivable balances
Effect:  Visual warning - money owed
Bold:    Yes (emphasis)
```

### **Credit Amount (Green Text)**
```
Hex:     #27AE60
RGB:     39, 174, 96
Usage:   Credit amounts, payable balances
Effect:  Positive indicator - money received
Bold:    Yes (emphasis)
```

### **Text Color (Dark Blue-Gray)**
```
Hex:     #2C3E50
RGB:     44, 62, 80
Usage:   Main text, body content
Effect:  Professional, readable, not pure black
Contrast: High contrast on white (AAA standard)
```

### **Secondary Text (Light Gray)**
```
Hex:     #7F8C8D
RGB:     127, 140, 141
Usage:   Labels, metadata, secondary information
Effect:  De-emphasis of less important information
Contrast: Good contrast but softer than main text
```

- ✅ Page break in correct position

## 🎓 Design Principles Applied

### **Visual Hierarchy**
```
Largest (80pt):    Company name - Most important brand
Large (48pt):      Main title - Document purpose
Medium (24pt):     Section headers - Content organization
Normal (20pt):     Body text - Main content
Small (18pt):      Labels - Supporting info
Smallest (16pt):   Metadata - Least important
```

### **Color Psychology**
```
Red (#E8312B):     Creates sense of importance, financial matter
Green (#27AE60):   Positive, success, money received
Blue-Gray:         Professional, corporate, trustworthy
Gray:              Neutral, background, secondary
White:             Clean, professional, empty space
```

### **White Space**
```
Used for:    Visual breathing room, elegance, readability
Margins:     1 inch all sides (generous professional margins)
Padding:     Adequate cell spacing
Spacing:     Proper paragraph spacing (not cramped)
Effect:      Modern, professional, high-end appearance
```

### **Typography**
```
Hierarchy:   5-level clear visual hierarchy
Readability: 20pt+ body text (easy to read)
Contrast:    Dark text on light/colored backgrounds
Spacing:    Proper line height (1.5-1.8)
Alignment:   Clean left-aligned text, right-aligned numbers
```
