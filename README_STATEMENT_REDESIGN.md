## 🛠️ Technical Stack

- **Backend**: Node.js + Express.js
- **Frontend**: React with axios
- **Database**: MongoDB
- **Document Generation**: docx library
- **Format**: Word 2007+ (.docx)
- **Security**: JWT + Role-based access
- **Logging**: Audit service
