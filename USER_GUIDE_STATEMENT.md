## 📥 How to Download Your Statement

### **Step 1: Navigate to Statement Page**
- Log in to your customer account
- Go to **"Account"** → **"Statement"** (or **"Statements"** section)

### **Step 2: Select Date Range (Optional)**
- **From Date**: Select the start date for your statement
- **To Date**: Select the end date for your statement
- Leave blank to get the full year statement

### **Step 3: Download Your Preferred Format**

#### **Option A: Professional Word Document (Recommended) 📄**
- Click button: **"📄 Download Statement (Word)"**
- This will download a `.docx` file (Microsoft Word format)
- **Best for**: Printing, professional sharing, archiving
- **Features**: Beautiful formatting, color-coded data, professional layout

#### **Option B: Excel Spreadsheet (For Analysis) ⬇**
- Click button: **"⬇ Download (Excel)"**
- This will download an `.xlsx` file (Excel format)
- **Best for**: Data analysis, sorting, custom calculations
- **Features**: Sortable columns, filterable data, spreadsheet functions

### **Step 4: Save & Use**
- File will download automatically
- Filename format: `statement_[your-code]_[timestamp].docx` or `.xlsx`
- Open with Microsoft Word or any compatible application

## 💡 Tips & Best Practices

### **For Printing**
1. Download Word version
2. Open in Microsoft Word or Google Docs
3. Ensure printer is set to color for best results
4. Print on A4 paper, portrait orientation
5. Use standard margins

### **For Record Keeping**
1. Download PDF version (from Word): File → Export as PDF
2. Save with clear naming: `statement_2026-05-02_to_2026-05-31.pdf`
3. Store in organized folder structure by year and month
4. Back up digitally and physically

### **For Analysis**
1. Download Excel version
2. Use Excel filters and formulas
3. Create pivot tables for deeper analysis
4. Export calculations back to management

### **For Sharing**
1. Download Word version
2. Share via email or secure file transfer
3. Can be opened by anyone with Word/Google Docs
4. Professional appearance suitable for auditors

## ❓ Frequently Asked Questions

### **Q: What's the difference between Word and Excel formats?**
**A:** 
- **Word (.docx)**: Professional, formatted, easy to print, suitable for official use
- **Excel (.xlsx)**: Data-focused, sortable, allows calculations, good for analysis

### **Q: Can I edit the downloaded statement?**
**A:**
- **Word**: Yes, but we recommend not changing amounts or dates
- **Excel**: Yes, for analysis purposes, but keep original copy

### **Q: How far back can I go?**
**A:** You can download statements for any date range. Default is the entire current year.

### **Q: Is my data secure?**
**A:** 
- All downloads are encrypted
- Logged with timestamp and your user ID
- Only you can download your statement
- Cannot be accessed by other customers

### **Q: What if the dates don't match?**
**A:** Make sure you're in Pakistan timezone (Asia/Karachi). Dates are formatted as DD-MMM-YYYY (e.g., 02-May-2026).

### **Q: Can I download multiple statements?**
**A:** Yes! Download and save as many as you need.

### **Q: What format is best for my accountant?**
**A:** Word format (.docx) - it's professional, complete, and prints beautifully.

## 🆘 Need Help?

### **Common Issues**

**Problem**: "File didn't download"
**Solution**: 
- Check your internet connection
- Verify pop-ups aren't blocked
- Try a different browser

**Problem**: "Can't open .docx file"
**Solution**:
- Install Microsoft Word or Google Docs
- Or use online tools like Office365.com
- Online converter services available

**Problem**: "Numbers don't look right"
**Solution**:
- Check your date range selection
- Ensure account code is correct
- Contact support with screenshots

**Problem**: "Document looks different when printed"
**Solution**:
- Use Print Preview before printing
- Adjust printer settings (margins, scaling)
- Use 100% scale (not "Fit to Page")
